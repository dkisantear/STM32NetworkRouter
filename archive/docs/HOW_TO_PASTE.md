# How to Paste in SSH Terminal

## Quick Methods (Try These):

### Method 1: Right-Click (Easiest)
- **Right-click** in the terminal window
- The text should paste automatically!

### Method 2: Keyboard Shortcuts
- **Ctrl + Shift + V** (most common in Windows)
- **Shift + Insert** (works in many terminals)
- **Ctrl + V** (sometimes works)

### Method 3: Terminal Menu
- Look for a **Paste** option in the terminal menu (usually Edit â†’ Paste)

---

## What to Paste

Open the file **`EXECUTE_THIS_ON_PI.sh`** in your editor (it's in the project directory), select **ALL** the contents (Ctrl+A), copy (Ctrl+C), then paste into the Pi terminal using one of the methods above.

---

## Pro Tip

If pasting doesn't work:
1. Make sure you're **clicked inside** the terminal window
2. Try **right-clicking** first (easiest!)
3. If that doesn't work, try **Ctrl+Shift+V**

---

## After Pasting

Press **Enter** to execute the script. It will run automatically and set everything up!
